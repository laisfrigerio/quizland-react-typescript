# QuizLand Frontend App

QuizLand is Quiz frontend application made with React

## Stack

- Node.js 22
- TypeScript 5
- Jest
- Vite
- quizland-core npm package

## Features

- [x] Theme (light & dark mode)
- Select Quizzes
- Choose one option
- Choose multiple options
- Set Timer
- Trainer and Exam Mode
